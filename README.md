# Smooosie
make a JUCE project with
* cmake,
* catch2 and
* trompeloeil

# What is where?
Find the application code in `src`.
The tests in `tests`.
`Catch2`, `JUCE` and `trompeloeil` in `submodules`.

# How to run this
In your terminal, navigate to the root folder. Enter:
* `cmake -S . -B build`
* `cmake --build build`
